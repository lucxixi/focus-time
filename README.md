# Memory Focus Timer

## 项目简介

Memory Focus Timer（专注计时器）是一款帮助用户记录和管理专注时间的 Web 应用。通过简洁的三栏布局设计，用户可以轻松管理长期事项、阶段事项，并通过计时器记录每一轮的专注时间。

## 核心功能

### 1. 事项管理

- **长期事项**：管理持续性的习惯或长期目标（如学习英语、健身等）
- **阶段事项**：管理短期项目或阶段性任务
- 支持事项的新增、编辑、删除和完成标记
- 已完成事项自动归档，可折叠查看

### 2. 专注计时

- **任务输入**：每轮开始前输入本轮要做的任务
- **实时计时**：精确记录专注时长（时:分:秒）
- **状态显示**：清晰的状态指示器（未开始/进行中/已暂停）
- **今日累计**：实时显示当日总专注时长

### 3. 有效性验证

每轮结束后进行两步验证：

1. **初步判断**：这一轮是有效专注吗？（有效 / 无效）
2. **精细校准**：如果部分有效，可选择有效比例（几乎无效 / 一半有效 / 大部分有效）

### 4. 历史记录

- **今日记录**：查看今天所有专注轮次的详细信息
- **历史汇总**：按日期查看历史记录，支持折叠展开
- **统计信息**：每天显示总时长、有效时长、轮数等统计数据

### 5. 个性化背景

- 支持上传自定义背景图片
- 背景保存在本地，刷新后依然保留
- 可一键恢复默认背景

### 6. 响应式缩放适配

- **整体等比缩放**：基于 1200px 设计稿自动缩放
- **Notion 嵌入支持**：完美适配 Notion 等外部平台嵌入场景
- **缩放范围**：0.75x - 1.0x，确保字体清晰可读
- **流畅响应**：窗口变化实时调整缩放比例

## 技术栈

- **前端框架**：原生 JavaScript + HTML + CSS
- **构建工具**：Vite 5.x
- **数据持久化**：localStorage（本地存储）
- **响应式方案**：CSS Transform Scale

## 使用指南

### 1. 创建事项

- 点击"长期事项"或"阶段事项"右侧的 **+** 按钮
- 在弹出框中输入事项名称
- 点击"创建"按钮

### 2. 开始专注

1. 在"这一轮我要做什么？"输入框中输入任务内容
2. 点击"开始"按钮
3. 专心工作，计时器自动记录时间
4. 完成后点击"结束"按钮

### 3. 验证有效性

1. 选择本轮是否有效（✓ 有效 / ✗ 无效）
2. 如果选择无效，进一步选择有效比例
3. 系统自动记录并更新统计数据

### 4. 继续或结束

- **再来一轮**：继续下一轮专注
- **今天结束**：查看今日汇总数据

### 5. 查看历史

- 点击右侧"历史记录"展开查看
- 点击日期左侧的箭头展开/折叠该日的详细记录
- 已完成事项会在左侧边栏底部的"已完成事项"中显示

## 数据说明

### 本地存储

所有数据保存在浏览器的 localStorage 中，包括：

- `focusRecords`：专注记录
- `habits`：长期事项
- `projects`：阶段事项
- `background`：背景图片（Base64 编码）

### 数据安全

- 数据仅保存在本地浏览器中
- 清除浏览器数据会导致记录丢失
- 建议定期导出重要数据（功能开发中）

## 设计特点

### 三栏布局

- **左栏（30%）**：事项管理区域
- **中栏（42%）**：计时器主功能区
- **右栏（25%）**：历史记录查看区

### 视觉设计

- 简洁清爽的界面风格
- 柔和的渐变背景
- 半透明卡片设计
- 流畅的过渡动画

### 响应式设计

- 采用整体缩放方案，保持布局比例
- 适配 Notion 等嵌入场景
- 窗口小于 1200px 时自动缩放
- 最小缩放 0.75x，确保可读性
