# 火华的专注计时器

## 项目简介

一款帮助记录和管理专注时间的 Web 应用。三栏布局，左侧管理事项，中间计时，右侧查看记录。数据云端同步，换设备不丢失。

## 核心功能

### 1. 事项管理

- **长期事项**：管理持续性习惯或长期目标（如学习英语、健身等）
- **阶段事项**：管理短期项目或阶段性任务
- 支持新增、编辑、删除和完成标记
- 已完成事项自动归档，可折叠查看
- 今日 Top 3 专注事项自动汇总展示

### 2. 专注计时

- 选择一个事项后开始计时，精确记录时长（时:分:秒）
- 支持填写本次专注内容备注
- **随便开始**：5 分钟快速计时模式，倒计时进度条可视化
- 今日累计专注时长实时显示

### 3. 今日素材计数

- 右侧面板内置素材计数器（目标 3 个）
- 点击 +1 累计，进度条可视化，达标后提示

### 4. 历史记录

- **今日记录**：右侧面板实时展示当天所有专注轮次
- **全部历史**：点击"全部历史"按钮，从右侧滑出历史记录抽屉
  - 按日期分组，显示每日累计时长
  - 支持按日期筛选
  - 今天、昨天自动显示中文标签
  - 支持 Esc 或点击遮罩关闭

### 5. 个性化背景

- 支持上传自定义背景图片
- 背景保存在本地，刷新后依然保留
- 可一键恢复默认背景
- 背景图模式下界面自动切换为暗色半透明风格

### 6. 响应式缩放适配

- 基于 1200px 设计稿整体等比缩放
- 适配 Notion 等外部平台嵌入场景
- 缩放范围：0.75x – 1.0x

## 技术栈

- **前端**：原生 JavaScript + HTML + CSS
- **构建工具**：Vite
- **数据持久化**：Supabase（云端数据库）
- **响应式方案**：CSS Transform Scale

```bash
npm install
npm run dev
```

## 数据说明

所有专注记录、事项数据通过 Supabase 云端存储，本地背景图片仍使用 localStorage。
